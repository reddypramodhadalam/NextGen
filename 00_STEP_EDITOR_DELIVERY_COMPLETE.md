```

├── Tabs (Details | Steps | Advanced)
├── StepEditor Component
│   ├── Insert Dialog
│   ├── Step Cards
│   ├── Edit Mode
│   └── Action Buttons
└── Validation Layer
    ├── Form validation
    ├── Step validation
    └── Error handling
```

### State Management
```
useStepManagement Hook
├── steps: TestStep[]
├── insertStep()
├── deleteStep()
├── updateStep()
├── moveStepUp()
├── moveStepDown()
├── reorderSteps()
├── getStep()
├── getAllSteps()
├── validateStep()
├── exportSteps()
├── importSteps()
├── duplicateStep()
```

### Data Flow
```
User Action → Component Handler → Hook Method
→ Auto-renumbering → State Update → Callback
→ Parent Update → Database Save
```
