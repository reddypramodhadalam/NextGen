## ⚠️ IF IT ASKS FOR MANUAL UPDATE

The script might ask you to manually update one file.

**It will tell you exactly what to do:**

1. Open file: `client/src/pages/TestCaseEditPage.tsx`
2. Find old component: `<TestCaseForm ... />`
3. Replace with: `<EnhancedTestCaseEditor ... />`
4. Save file
5. Press Enter in script

## 🆘 IF SOMETHING FAILS

The script will tell you exactly what's wrong.

**Usually it's one of these:**

1. **Files not found**: Copy 3 files to project root
2. **npm not installed**: Download from nodejs.org
3. **git not installed**: Download from git-scm.com
4. **TypeScript errors**: Fix errors, run script again
5. **Wrong directory**: Make sure you're in project root
